## Templates

### Webpack-simple
- Plantilla de Vue.js 
- [https://github.com/vuejs-templates/webpack-simple](https://github.com/vuejs-templates/webpack-simple)

- **Como usar**
```
$ npm install -g vue-cli
$ vue init webpack-simple my-project
$ cd my-project
$ npm install
$ npm run dev
```
